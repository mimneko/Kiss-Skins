# Kiss-Skins
Wordpressテーマ[Cocoon](https://wp-cocoon.com/)用オリジナルスキン「Kiss」シリーズ。
シンプルながらも、オシャレで機能的なデザインを追求しました。
各ディレクトリにファイルを置いています。

# ラインナップ（全10種）
- Kiss-Red
- Kiss-Green
- Kiss-Blue
- Kiss-Yellow
- Kiss-Purple
- Kiss-Pink
- Kiss-Orange
- Kiss-Brown
- Kiss-Gray
- Kiss-Black
